What is Mini Pupper?
^^^^^^^^^^^^^^^^^^^^^

Mini Pupper is a low-cost, personal quadruped kit with open-source software. Mini Pupper was created by MANGDANG in Septemper 2021. With Mini Pupper, youâ€™ll be able to build a robot dog that can drive around, and have enough horsepower to create exciting applications.

For more information, please refer to `Mini Pupper Kickstarter Campaign <https://www.kickstarter.com/projects/336477435/mini-pupper-open-sourceros-robot-dog-kit>`_, `Mini Pupper Makuake Campaign <https://www.makuake.com/project/mini_pupper/>`_, `Mini Pupper 2 Kickstarter Campaign <https://www.kickstarter.com/projects/336477435/mini-pupper-2-open-source-ros2-robot-kit-for-dreamers>`_,  `Amazon AWS Robotics Blog <https://aws.amazon.com/blogs/robotics/build-and-simulate-a-mini-pupper-robot-in-the-cloud-without-managing-any-infrastructure/>`_, `Amazon 2022 re:MARS Workshop Guide <https://aws.amazon.com/blogs/robotics/build-and-simulate-a-mini-pupper-robot-in-the-cloud-without-managing-any-infrastructure/>`_. 

Many makers are contributing to Mini Pupper project, some parts referred to other contributors' documents, many thanks for all the guys who helped Mini Pupper project!
